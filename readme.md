# WORKOUT TRACKER
Aplikasi untuk membuat dan melacak kegiatan olahraga

## Fitur :
- Membuat Nama Workout
- Durasi Waktu yang di inginkan
- Jeda Istirahat
- Daftar Workout
- Simpan Preset berdasarkan Daftar Workout
- Akses Ke Preset Workout

## Engine yang digunakan :
- HTML
- JAVASCRIPT
- CSS
